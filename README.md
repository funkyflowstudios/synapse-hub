## 🔧 Quick Start

1. **Setup Development Environment**
   ```bash

   npm run setup:dev-env
   ```

2. **Configure AI IDE (Optional)**
   - Restart Cursor IDE to apply Claude 4 Sonnet settings

   ```bash
   npm run dev:all
   ```

4. **Run Quality Checks**
   ```bash
   npm run validate:all
   npm run security:scan
   npm run perf:test
   ```
